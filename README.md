# Money Tracker Bot

Telegram Bot for tracking your money.

## Feature

- Record expense
- Edit category & date
- Delete expense

## How to run

### Docker

1. Register new bot in BotFather
2. Copy `.env.example` to `.env`
3. Fill `.env` with your token, bot username and telegram account id
4. Run `docker-compose up`
